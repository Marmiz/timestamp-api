Timestamp Microservice API.
A project made for the Free Code Camp curriculum.

What's this?

This is a microservice API that will accept either a human readable data format or a unix formatted date and will output a json with both vaues formatted.

Usage.

1) User can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date formatted as 'MMMM DD YYYY'
(example: January 1 2016)
2) If it does, it returns both the Unix timestamp and the natural language form of that date.
3) If it does not contain a date or Unix timestamp, it returns null for those properties.

Examples of usage.

manually typing the date after the URL or entering the unix value.
Example(december 25 2015) 

https://timestampfreecodecamp.herokuapp.com/december%2025%202015
https://timestampfreecodecamp.herokuapp.com/1451001600
https://timestampfreecodecamp.herokuapp.com/12%20december

Example of output.

        {
         humanReadable: "January 10, 2015",
         unix: "1420848000"
        }

        {
        humanReadable: "null",
        unix: "null"
        }
